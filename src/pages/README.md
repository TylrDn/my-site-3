## ğŸ¨ Styling Strategy

All UI styling is class-based and managed in `globalStyles.js`. This ensures:

- Consistent visual system across the site
- Responsive layouts (flex/grid)
- No inline style repetition

### Example Usage

```js
<div class="card">
  <h3>Title</h3>
  <p>Description here</p>
</div>
````

Injected using:

```js

```
